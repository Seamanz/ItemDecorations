# Installation
To get a Git project into your build:

**Step 1.** Add the Jcenter repository to your build file
```
allprojects {
    repositories {
        ...
       jcenter()
    }
}
```
**Step 2.** Add the dependency
```
dependencies {
    implementation 'com.github.zane.itemdecorations:item-decorations:0.0.2'
}
```
